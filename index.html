My Skill

HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Fungsi:
✅ Membuat elemen-elemen dasar halaman web seperti teks, gambar, tombol, tabel, dan form.
✅ Menentukan hirarki konten (judul, paragraf, daftar, dll.).
✅ Menghubungkan berbagai halaman dengan hyperlink.

CSS

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web.
Fungsi:
✅ Mengatur warna, font, ukuran teks, dan tata letak halaman.
✅ Membuat halaman lebih menarik dengan animasi dan efek visual.
✅ Membantu dalam desain responsif agar website tampil baik di berbagai perangkat.

JAVA SCRIPT

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas pada halaman web.
Fungsi:
✅ Membuat efek animasi, pop-up, dan validasi form.
✅ Memanipulasi HTML dan CSS secara dinamis (DOM Manipulation).
✅ Menghubungkan website dengan server untuk mengambil data (AJAX, API).

JAVA

Java adalah bahasa pemrograman yang bersifat Object-Oriented dan dapat dijalankan di berbagai platform.
Fungsi:
✅ Digunakan untuk membangun aplikasi web, mobile (Android), dan desktop.
✅ Banyak digunakan dalam pengembangan backend dan sistem enterprise.
✅ Digunakan untuk membuat aplikasi yang membutuhkan keamanan tinggi seperti perbankan.

BOTSTRAP

Bootstrap adalah framework CSS dan JavaScript yang digunakan untuk membuat tampilan website dengan cepat dan responsif.
Fungsi: ✅ Membantu dalam pembuatan desain web yang responsif (cocok di semua perangkat).
✅ Menyediakan komponen siap pakai seperti navbar, tombol, form, dan modal.
✅ Menghemat waktu dalam pengembangan website karena menggunakan grid system.
